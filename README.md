# Harsha Tattoo Studio

Official website for **Harsha Tattoo Studio**, Coimbatore.

## Project Structure

```
harsha-tattoo/
├── index.html          # Main HTML (structure only)
├── css/
│   └── styles.css      # All styles, variables, animations
├── js/
│   └── main.js         # Cursor, scroll reveal, form, navbar
└── images/
    ├── logo.png
    ├── hero-bg.jpeg
    ├── gallery-1.jpeg  → gallery-9.jpeg
    └── logo-footer.png
```

## Editing Guide

| What to change | Where |
|---|---|
| Colors / fonts | `css/styles.css` → `:root` variables |
| Sections / content | `index.html` |
| Animations | `css/styles.css` → `@keyframes` |
| Cursor / scroll / form logic | `js/main.js` |
| Gallery images | Replace files in `images/` (keep same filenames) |

## Live Preview

Open `index.html` in any browser — no build step required.
